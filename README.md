# tiki-ticket

A ticket box demo for Tiki Apply

# Installation

```sh
git clone https://github.com/khanhhaquang/tiki-ticket.git
```

Then

```sh
npm install
```

# Run

```sh
npm start
```

The web will be opened automatically at [localhost:3000](http://localhost:3000)

# Test

```sh
npm test
```

# Build

```sh
npm run build
```
